Introduction
--------------------------------------------------------------------------------

**yehua** A command line tool to auto generate a pyexcel related python package


Installation
--------------------------------------------------------------------------------

You can install it via pip:

.. code-block:: bash

    $ pip install yehua


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/pyexcel/yehua.git
    $ cd yehua
    $ python setup.py install
