Introduction
--------------------------------------------------------------------------------

**yehua** is a command line tool to provide a default scaffolding for a python package.


Installation
--------------------------------------------------------------------------------

You can install it via pip:

.. code-block:: bash

    $ pip install yehua


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/chfw/yehua.git
    $ cd yehua
    $ python setup.py install
